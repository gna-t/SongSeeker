# SongSeeker

A music discovery app that shows you one song at a time so you can find new music without scrolling through endless lists.

## What it does

- Shows one track at a time  
- Lets you save or skip songs  
- Explore more from the same artist  
- Keeps a list of saved songs  

## Why I built this

Most music apps overwhelm you with choices. I wanted something simple where you can quickly decide if you like a song and move on.

## Live Demo

https://SongSeek.vercel.app/

## How to run

Clone the repo and open index.html  
or run with a simple local server

## Tech used

- HTML, CSS, JavaScript  
- iTunes Search API  
- LocalStorage  

## What I worked on

- Designed the swipe-style discovery experience  
- Built the save, skip, and explore features  
- Integrated the iTunes API for real music previews  
- Added fallback logic to keep the app working smoothly  
- Used AI to speed up development and refine UX  

## AI workflow

- Used structured prompts to build features step by step  
- Focused on user flow before writing logic  
- Iterated on UI and behavior using AI suggestions  
- Debugged and corrected issues manually  

## What I learned

- Simplicity makes apps more engaging  
- AI helps move fast, but still needs direction  
- Good UX comes from testing and refining small details  

## Notes

This project focuses on interaction. The goal is to make music discovery feel quick and natural instead of overwhelming.
